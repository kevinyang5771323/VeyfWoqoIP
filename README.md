# 宠物管理系统 java100

## 项目概述

宠物管理系统是一个基于B/S架构的MVC模式管理系统，采用Java和SpringBoot作为后端开发，结合Vue实现前端界面。系统实现前后端分离，适用于宠物领养、救助及相关信息管理。

## 技术栈

- **后端**：Java、SpringBoot、Mybatis-Plus
- **前端**：Vue
- **数据库**：Mysql
- **开发工具**：IDEA、JDK1.8、Maven

## 系统角色

系统包含两个主要角色：用户和管理员。

### 用户端功能

- **首页**：展示公告列表、宠物科普、流浪宠物介绍、热门活动。
- **宠物领养**：搜索、申请领养宠物，查看领养进度。
- **宠物救助**：查看需要救助的流浪宠物，上报新流浪宠物信息。
- **丢失宠物查看**：查看走失宠物和失主信息。
- **宠物论坛**：交流领养心得，参与话题讨论。

### 管理员端功能

- **主页**：展示用户注册数据统计。
- **用户管理**：管理用户信息。
- **角色管理**：为用户分配系统菜单权限。
- **菜单管理**：管理系统后台菜单。
- **文件管理**：统一管理文件。
- **流浪动物管理**：发布流浪宠物信息。
- **待绝育动物管理**：管理流浪动物健康信息。
- **申请领养管理**：处理领养申请。
- **评论管理**：管理论坛评论。
- **流浪动物救助**：发布和更新救助信息。
- **喂养点管理**：管理宠物喂养点。
- **动物走失管理**：管理丢失宠物信息。
- **救助站管理**：管理救助站点信息。
- **帖子管理**：管理论坛帖子。
- **捐赠管理**：管理捐赠信息。
- **公告管理**：管理首页公告。
- **科普文章管理**：管理科普文章。

## 部署步骤

1. 准备好Java、Mysql等运行环境。
2. 导入数据库文件，初始化数据库。
3. 编译后端代码，部署至服务器。
4. 编译前端代码，部署至Web服务器。
5. 根据配置文件（如`application.properties`）配置系统参数。
